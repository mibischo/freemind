void freemind.extensions.PermanentNodeHookAdapter.onAddChild ( MindMapNode  addedChildNode)

Is called if the addedChildNode is inserted as a direct child of the node, this hook is attached to. The cases in which this method is called contain new nodes, paste, move, etc.

Ah, don't call propagate in this method, as paste introduces nodes with the hook and you'll have them twice, ... see onNewChild
